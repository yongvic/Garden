# Garden — Product Definition

## What it is
International B2B marketplace for booking professional event spaces, conference rooms, accommodations, and equipment. Primary markets: West Africa (FCFA) with English/French support.

## Audience
- **Guests (CUSTOMER):** Event planners, agencies, startups, corporate teams needing short-term spaces.
- **Hosts (LANDLAND):** Property owners, venue managers, equipment rental businesses.
- **Admins:** Platform operators moderating listings and disputes.

## Register
product — design serves the product (marketplace + dashboards).

## Core flows
1. Browse → filter → view listing (public)
2. Register/login → request booking → pay (Stripe XOF) → host confirms
3. Host: create listing → manage availability → confirm bookings
4. Post-stay: verified reviews, damage claims

## Brand voice
Confident, specific, professional. No buzzwords. French default, English secondary.

## Design direction
Premium light-first SaaS. Forest green primary (Garden), warm amber accent, generous whitespace, no glassmorphism defaults, no blue-purple gradients.
